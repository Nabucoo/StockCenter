# 📦 StockCenter

Sistema web de controle de estoque desenvolvido para a disciplina de **Engenharia de Software II**.

O StockCenter tem como objetivo facilitar o gerenciamento de produtos e movimentações de estoque, oferecendo uma solução prática para controle de entradas, saídas e monitoramento de informações relevantes para a tomada de decisões.

## 🚀 Funcionalidades

- Cadastro, edição e exclusão de produtos;
- Controle de entrada e saída de estoque;
- Registro e consulta de movimentações;
- Gestão de usuários;
- Dashboard com métricas e indicadores;
- Consulta do histórico de operações.

## 🛠️ Tecnologias Utilizadas

### Frontend
- React
- Vite
- JavaScript

### Backend
- Node.js
- Express

### Banco de Dados
- MySQL
